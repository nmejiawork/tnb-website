OpenClaw is a free, open-source AI agent that runs on your own machine. Unlike a chatbot in a browser tab, it connects directly to apps, browsers, and system tools through a modular system of skills. You bring your own API key (Claude, GPT, an open-source model, whatever you have access to) and the agent uses it to actually do things on your computer rather than just chat about them.

The project came from Austrian developer Peter Steinberger, originally launched as Clawdbot in November 2025. After trademark complaints from Anthropic in late January 2026, it was renamed to Moltbot, then renamed again to OpenClaw three days later. The rebrand coincided with going viral: 100K GitHub stars in under 48 hours, the fastest growth in GitHub's history. By March 2026 the repo had crossed 247K stars and 47.7K forks.

OpenClaw became central to a public dispute over how AI agents access Claude. Because OpenClaw lets users plug their Claude Code subscription into a third-party agent, Anthropic announced subscribers would need to pay extra to use it. The relationship escalated in April 2026 when Anthropic temporarily banned Steinberger from Claude entirely, then restored access. The conflict has become the canonical example of a tension every AI coding tool now navigates: who can use whose models, under what terms, in what wrapper.

For builders, OpenClaw matters as a category bet. Cursor and Claude Code are IDE-style products with opinions about how you work. OpenClaw is an extensible agent runtime, closer to a Linux-shell-with-AI than to a polished editor. Steinberger joined OpenAI in February 2026 and a non-profit foundation now stewards the project, decoupling it from any single company.
